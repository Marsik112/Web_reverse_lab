
Web Reverse Lab — это REST API-сервис для загрузки и первичного анализа бинарных файлов.

Проект позволяет загружать исполняемые файлы, сохранять информацию о них в SQLite, выполнять базовый анализ через Linux-утилиты и получать результаты через API.

Возможности

- загрузка бинарных файлов;
- проверка расширения файла;
- ограничение размера файла;
- сохранение файла в директорию "uploads";
- сохранение метаданных в SQLite;
- получение списка загруженных файлов;
- получение информации о файле по ID;
- удаление файла;
- базовый анализ через "file";
- извлечение строк через "strings";
- вычисление SHA256;
- кэширование результатов анализа в БД.

Запуск проекта

Создать виртуальное окружение:

python -m venv .venv

Активировать окружение:

source .venv/bin/activate

Установить зависимости:

pip install -r requirements.txt

Запустить сервер:

uvicorn app.main:app --reload

После запуска API будет доступно по адресу:

http://127.0.0.1:8000

Swagger-документация:

http://127.0.0.1:8000/docs

API Endpoints

Проверка работы сервера

GET /

Загрузка файла

POST /files

Получение списка файлов

GET /files

Получение информации о файле

GET /files/{file_id}

Получение анализа файла

GET /files/{file_id}/analysis

Удаление файла

DELETE /files/{file_id}
